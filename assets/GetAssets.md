# How to get the assets
You have to download the [Post Apocalypse Pixel Art Asset Pack](https://thelazystone.itch.io/post-apocalyptic-pixel-art-asset-pack) by [TheLazyStone](https://thelazystone.itch.io/).

Unpack the .zip in the project folder under `./assets/PostApocalypse_AssetPack` 

The folder structure should look something like that 
```
GPROG-PROJECT/
├── assets/
│   └── PostApocalypse_AssetPack/
│       ├── Character/
│       ├── Enemies/
│       ├── Objects/
│       ├── Tiles/
│       ├── UI/
│       └── LICENSE.txt
├── docs/
└── src/
```
